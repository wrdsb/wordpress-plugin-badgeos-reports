The "BadgeOS Reports Add-on" adds achievement analytics and reports to BadgeOS.

== Description ==

The "BadgeOS Reports Add-on" is a reporting and analytics tool for BadgeOS. It adds a new Reports menu for BadgeOS admin users and automatically creates a dynamic Report for all Achievement types, Badges, and Users.

Each Achievement and Badge Report includes:

* Achievement name, earning count, date last earned
* Summary data: total number of achievements created, total number awarded, average earning per achievement
* Drill down to detailed achievement list of each user who has earned the specified achievement, including their username, first and last name, earned count, and date last earned

User Report:

* Includes username, first and last name, total number of achievements earned, date of last achievement activity and date user joined the site
* Summary data: Total users and new users joining the site during specified data period

Additional Report features:

* Every report can be downloaded as a CSV file and opened in common tools like Excel or GoogleDocs.
* Limit each report's results to a specified date range
* Real-time filterable search of Report data
* Sortable column headers

Note: This Add-on requires the free BadgeOS plugin. [Get the BadgeOS plugin](http://wordpress.org/extend/plugins/badgeos/ "BadgeOS").

== Installation ==


1. First get and activate the free [BadgeOS plugin](http://wordpress.org/extend/plugins/badgeos/ "BadgeOS").
2. Then upload the 'badgeos-Reports-Add-on' plugin via the 'Plugins' menu in WordPress (or upload to the '/wp-content/plugins/' directory).
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Visit the "BadgeOS Reports" menu in the admin area of your site to run and export reports.
